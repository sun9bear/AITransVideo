# WEB_CONSOLE_PHASE2_LIGHT_PAGES_ACCEPTANCE_NOTE.md

## 1. 本阶段完成了什么

本阶段已完成第二批页面（轻量版）收口：

- “我的项目（轻量版）”
- “设置（极简版）”

并保持：

- `internal` 只作为弱入口保留
- 第一批页面与 review phase 1 基线未被破坏

## 2. 当前“我的项目（轻量版）”具备的能力

- 基于现有 job 列表快照展示历史任务
- 展示任务状态
- 展示最近更新时间
- 可进入项目详情页

## 3. 当前“设置（极简版）”具备的能力

- 展示当前版本说明
- 展示当前稳定基线说明
- 展示当前限制说明
- 展示少量默认项说明
- 如涉及 TTS 默认模型，当前只做说明，不提供编辑

## 4. `internal` 当前如何保留

`internal` 当前只作为弱入口保留。
本阶段没有把它扩张成独立页面族，也没有在其中承诺成体系功能。

## 5. 当前明确边界

- 不做 `voice_review` 原生前端化
- 不做登录 / 注册
- 不做数据库
- 不做多用户
- 不做商业化页面
- 不做复杂搜索 / 筛选
- 不做批量操作
- 不做配置中心
- 不做 `internal` 页面族

## 6. 后续推进前的前提

如果后续还要继续推进前端页面扩张或新能力，必须先重新定义下一阶段范围，再开始开发。
