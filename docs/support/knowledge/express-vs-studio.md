# Express 与 Studio 模式

爱译视频提供两种处理模式，用户在创建任务时选择。

## Express（快捷版）

- 自动跑完整流程，速度快、成本低。
- 不进入逐句审核，处理完成即交付。
- 适合：先验证效果、批量出片、对配音质量要求不高的内部使用。
- 音色：自动匹配预设音色。

## Studio（工作室版）

- 在工作台逐句复核译文、修改翻译。
- 可手动选择中文配音音色（多 provider 备选）。
- 在合法授权前提下支持声音克隆能力。
- 适合：需要发布或交付客户的内容。

## 怎么选

| 场景 | 推荐 |
| --- | --- |
| 第一次试用 | Express 跑一条样片 |
| 批量出短视频 | Express |
| 视频要发布到公众号 / B 站 | Studio |
| 客户付费的成片 | Studio |
| 想用克隆音色 | Studio |

## 切换模式

任务创建时选择。已创建的任务无法在 Express 与 Studio 间互相切换；如需更高质量结果，可基于同一源视频「另存为新任务」用 Studio 重新跑一遍。
