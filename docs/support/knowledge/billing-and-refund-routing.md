# 账单 / 退款 / 隐私 / 版权 — 人工分流准则

这些场景**不**由 AI 客服直接处理。AI 可以解释公开政策和收集必要信息，但不能作出最终承诺；满足任一条件即直接转人工：

## 自动转人工的场景

- 付款成功但套餐未到账。
- 重复扣费。
- 退款申请。
- 发票 / 对账。
- 隐私删除请求 / GDPR 类似条款下的数据请求。
- 版权投诉 / 侵权举报 / 授权争议。
- 账号安全问题（怀疑被盗）。
- 手机号归属变更。

## AI 客服可以做什么

- 给出公开政策摘要（《服务条款》《退款政策》《隐私政策》的关键条款）。
- 引导用户到正确入口（账单页、设置页、联系页）。
- 收集必要信息（如订单号 / 任务 ID）方便人工接手。

## AI 客服不能做什么

- 承诺退款一定通过。
- 承诺人工多久一定回复。
- 判断用户素材是否一定合法。
- 替用户执行高风险账户操作（修改手机号、删除账户、退订套餐）。
- 在没有授权的情况下读取任务、订单、账单或项目内容。

## 工单流程

1. AI 客服识别敏感关键词（如「退款」「投诉」「工信部」），命中即转人工。
2. 系统把会话摘要 + 用户上下文写入 `support_handoff_requests`。
3. 系统发邮件给运营团队（默认收件人 `sxz999@proton.me`，可在 admin/support 改）。
4. 运营在邮件里直接回复用户。
5. 处理完后在 admin/support 标记 `closed`。
