# 剪映草稿导出

任务完成后可下载剪映草稿工程，导入剪映 App 后可以继续编辑字幕、配音和素材轨道，不必从零铺时间线。

## 下载位置

进入工作台 → 任务详情 → 结果页，找到「剪映草稿」下载项。

> 如果你刚刚修改过字幕或配音，建议在结果页重新点「生成剪映草稿」，避免下载到旧版本。

## 包含什么

- 中文字幕轨。
- 中文配音音轨（每段独立，可单独删 / 替）。
- 原视频画面轨（如果你启用了「保留原画」）。
- 简单的转场和缝合，足够你直接预览。

## 不包含什么

- 文字标题、表情包、贴纸等定制元素 — 需要你在剪映里自己加。
- 完整 BGM / 音效 — 需要你自己导入。

## 在剪映里怎么打开

1. 解压下载的 zip 到任意目录。
2. 打开剪映 App → 我的剪辑 → 三点菜单 → 导入草稿。
3. 选择刚才解压出来的目录里的 `draft_content.json`。
4. 进入剪映剪辑页面继续编辑。

## 故障排查

- 「剪映打不开」：先确认你下载的是完整 zip（不是中途中断的）。
- 「字幕错位」：可能是导出时还在生成中。回到任务结果页，等 3-5 秒后重新生成草稿。
- 「找不到剪映草稿下载项」：当前任务可能是 Express 模式且未启用草稿导出。Studio 模式默认包含。
