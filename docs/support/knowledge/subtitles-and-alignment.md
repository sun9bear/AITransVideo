# 字幕与对齐

爱译视频默认输出三种字幕：中文 / 英文 / 中英双语。所有字幕都做了句级时间戳。

## 字幕来源

字幕由两个阶段决定：

1. **转写**：从原视频音频提取英文文本和粗对齐时间戳。
2. **翻译 + 对齐**：英文 → 中文，并把中文片段对齐回原英文片段时间窗。

## 精对齐（Whisper 对齐）

部分套餐 / 模式下，字幕会经过 Whisper 二次精对齐——把中文配音的实际边界对回到帧级精度。开关位置：

- 用户侧：默认开启，可在结果页查看是否启用。
- 管理员侧：admin/settings 的 Whisper 区域。

## 字幕异常的常见原因

- **字幕和配音对不齐**：
  - 检查任务是否还在进行中（结果页有「字幕精对齐」状态）。
  - Studio 模式下，单段重生成后建议「重新对齐」一次再下载字幕。
- **字幕条数和原音频不一致**：
  - 转写阶段如果遇到长停顿或多人同时讲话，可能合并 / 拆分原英文片段。
  - 在编辑模式拆分对应片段后重新对齐即可。
- **下载的字幕里有英文残留**：
  - 这可能是源视频中包含听不懂的专有名词，AI 选择保留原文。可在工作台手动改译。

## 双语字幕格式

`.srt` 格式，每条字幕的内容是「英文\n中文」两行。如果只想要单语字幕，下载时选择对应版本。
