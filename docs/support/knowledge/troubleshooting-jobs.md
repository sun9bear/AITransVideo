# 任务失败 / 异常排查

爱译视频会把任务失败的原因归类到几个 `error_category`。AI 客服根据 `error_category` 返回对应的模板化排查步骤。

## 常见错误类别与处理

### `source_unavailable`

源视频拉取失败。

- YouTube 链接被原作者下架或区域限制 → 换链接。
- 上传的本地视频损坏 → 重新上传完整文件。
- 网络拉取超时 → 稍后重试。

### `transcription_failed`

音频转写失败。

- 视频中没有清晰人声 → 提交一段有清晰人声的视频。
- 多人同时讲话且没有明显说话人区分 → 在 Studio 模式手动指定说话人数后重试。

### `translation_failed`

LLM 翻译阶段失败。

- 上游 LLM 临时不可用 → 稍后重试。
- 文本中含有触发内容审核的片段 → 检查原视频是否包含敏感词，必要时联系客服。

### `tts_failed`

中文配音合成失败。

- 选择的音色配额已用完 → Studio 切换音色。
- 文本过长触发单次合成上限 → 在编辑模式拆分超长片段。

### `alignment_failed`

音视频对齐失败。

- 通常是临时性问题。在任务详情页点「重新对齐」即可，不会重复扣额度。

### `publish_failed`

成片发布阶段失败。

- 临时存储问题或 R2 上传异常 → 稍后重试。
- 反复失败请联系客服并附上任务 ID。

## 通用规则

- **失败不计费**：任务因系统失败未生成可下载结果时，不扣额度。
- **不重复扣费**：同一任务的重试不会再次扣额度（除非「另存为新任务」时按新任务计算）。
- **保存任务 ID**：联系客服时务必提供任务 ID（在 URL `/workspace/{id}` 里）。

## 何时转人工

- AI 给的排查步骤试过仍然失败。
- 同一任务连续失败 3 次以上。
- 失败叠加付款 / 套餐到账问题。
- 用户明显不耐烦 / 投诉。
