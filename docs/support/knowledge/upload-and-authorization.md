# 视频上传与授权

## 支持的来源

- 本地视频文件上传（mp4 / mov 等常见格式）。
- YouTube 链接（适用于您本人频道或已获授权的视频）。
- 其他视频链接（站点支持差异较大，建议先小段试跑）。

## 授权要求（强制）

使用前必须确认对相关视频拥有合法授权。详细条款见站点《服务条款》页面。

不允许的来源（示例，不限于）：

- 未经授权的第三方版权视频。
- 包含未经允许的肖像 / 声音内容。
- 法律法规禁止传播的内容。

## 失败的常见原因

- YouTube 链接被原作者下架或区域限制。
- 上传的本地视频文件损坏或编码不兼容。
- 视频时长超过当前套餐上限（具体上限以套餐页为准）。
- 网络拉取超时。

失败的任务不会扣除额度。重试前先核实链接可访问、文件未损坏。

## 视频时长怎么算

按源视频实际时长统计。处理完成后计入当期额度；任务失败或主动取消不计费。

> 「不满意」不属于失败——已完成并交付的结果按时长正常计费，但可在工作台逐句修改并单段重生成，避免重复跑全片。
