# 产品基础

爱译视频是面向中文创作者的长视频翻译与中文配音工作台。聚焦海外英文长视频转中文配音版本——访谈、课程、播客、产品演示这类内容效果最稳定。

## 核心能力

- 海外英文长视频翻译为中文译文。
- 中文配音生成（多种音色、可选声音克隆）。
- 中文字幕（含双语 / 仅英文 / 仅中文）。
- 翻译文本导出。
- 原始素材包导出。
- 剪映草稿工程导出，下载后可在剪映继续精剪。

## 工作流概览

1. 创建任务：上传本地视频或粘贴 YouTube / 视频链接。
2. 任务处理：转写 → 翻译 → 配音 → 对齐 → 发布。
3. 审核与编辑：Studio 模式可逐句复核译文与音色。
4. 下载交付物：成片、音频、字幕、翻译文本、素材包、剪映草稿。
