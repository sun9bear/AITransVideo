# Web Console 结果区与入口分层指引

## 主结果
- 成品视频
- 完整配音音频
- 字幕文件

## 默认展示下载项
- 项目详情页默认先展示主结果下载项
- 当前任务页优先展示主结果是否已就绪，并在可用时引导进入项目详情统一查看与下载

## 次级 / 折叠下载项
- 翻译分段（JSON）
- 项目清单（JSON）
- 内部文件路径与补充产物

## 页面入口层级
- 当前任务：主入口是当前最该做的事；等待审核时进入对应 review，其余情况下进入结果与下载或项目详情
- 我的项目：已完成项目优先进入结果与下载；进行中项目优先回到当前任务；项目详情作为次级入口
- 项目详情：主入口优先是结果与下载或当前审核；日志、内部文件路径下沉到次级区
- 三类 review：主入口永远是提交当前审核；返回当前任务、查看项目详情属于次级入口
- 设置页：属于弱入口，只做说明，并把用户带回当前任务或我的项目

## 旧 Web UI
- 旧 Web UI / fallback 兼容能力仍保留在底层
- 旧入口不再作为最终用户的显式可见入口
